## 3. 데이터 구조 설계
(자 맞추기 게임의 데이터 구조는 정답 숫자, 현재 시도 횟수, 최대 시도 가능 횟수, 사용자 입력 기록, 게임 상태(진행 중/성공/실패) 등을 중심으로 구성하면 된다

**방법 1: 배열 사용 (기본)**
```c
// 예시: 학생 5명의 이름과 성적 저장
int answer;          // 정답 숫자
int attempts = 0;    // 현재 시도 횟수
int maxAttempts = 10;// 최대 시도 가능 횟수
int history[10];     // 사용자 입력 기록 저장
int gameState = 0;   // 0: 진행중, 1: 성공, 2: 실패


**방법 2: 구조체 사용 (선택)**
```c
struct Game {
    int answer;          // 정답
    int attempts;        // 현재 시도 횟수
    int maxAttempts;     // 최대 시도 횟수
    int history[10];     // 입력 기록
    int gameState;       // 게임 상태
};
struct Game game;         // 게임 데이터 저장
